# Programming Fundamentals Exercises

## Overview
This repo contains C++ solutions for various programming fundamentals exercises. It covers concepts like conditionals, loops, functions, and basic problem-solving.

## What You'll Learn
- Using `switch` statements for selective logic (e.g., day-of-week program)
- Basic math logic implementation (e.g., even/odd check, triangle type)
- Input/output handling in C++
- Translating simple problems into code

## Highlights
- **Days of the Week**: `switch` example mapping numbers to day names
- **Triangle Type**: Logic to classify triangles (equilateral, isosceles, scalene)
- More math and logic-based exercises
